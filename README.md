## Project Overview - FRD - Airline

This project involves the development of a Functional Requirements Document (FRD) for an airline loyalty program enhancement, focused on introducing a new promotional feature called *“Las Vegas Club”* within the airline miles system.

As a Business Analyst Trainee, I contributed to analyzing business requirements and documenting detailed functional specifications for system enhancements, including automated email notifications, user eligibility logic, and new web page integrations.

## Key Contributions

* Assisted in preparing the Functional Requirements Document (FRD) for a promotional feature in the airline miles program
* Documented business rules and eligibility criteria for “Las Vegas Club” profile creation and removal
* Defined functional requirements for:

  * Automated email notifications
  * User navigation and workflows
  * Promotional page enhancements
* Created structured documentation for new features such as:

  * Vegas Club page
  * Sweepstakes enrollment module
* Identified validation rules and data field requirements for user input forms
* Supported the design of user journeys and system interactions

## Key Features Covered

* Automated email triggers based on user eligibility
* Customer profile creation based on travel history
* Promotional web pages and navigation logic
* Sweepstakes enrollment system with validations
* Business rules for customer segmentation
